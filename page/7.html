Show HN: Usque – Open-Source Cloudflare Warp Masque Client

作者: MaryJohanna | 评分: 5 | 评论数: 0 | 发布时间: 2025-03-19 22:58
文章摘要

**Usque** 是一个开源项目，重新实现了 Cloudflare WARP 客户端的 MASQUE 协议。它支持多种模式，包括原生隧道模式（目前仅限 Linux）、SOCKS5 代理模式和 HTTP 代理模式。项目使用 Go 语言编写，支持多平台，包括 Linux、Windows 和 Android。用户可以通过源码构建、直接下载二进制文件或使用 Docker 部署该工具。主要功能包括注册新账户、设备 enrollment、以及通过不同模式暴露 WARP 服务。用户在使用前需进行注册，注册过程会自动处理设备登记和 MASQUE 密钥 enrollment。该项目适用于希望使用 MASQUE 协议代替传统 VPN 方案的用户。

SheepShaver is an open source PowerPC Apple Macintosh emulator

作者: janandonly | 评分: 135 | 评论数: 22 | 发布时间: 2025-03-16 10:32
文章摘要

SheepShaver是一款开源的PowerPC苹果Macintosh模拟器，可运行Mac OS 7.5.2至9.0.4，支持Windows、macOS和Linux系统。它被视为新版macOS中不再提供的Classic Environment的良好替代品。用户需自行提供ROM文件和Mac OS。网站提供各系统的安装指南及辅助文件，并设有支持论坛。SheepShaver最初于1998年作为BeOS系统的商业产品发布，2002年转为开源，由Christian Bauer开发，后由Gwenole Beauchesne继续维护至2008年，之后由社区贡献更新。
