Nvidia open-source nanite: continuous level of detail (LOD) mesh library

作者: klaussilveira | 评分: 81 | 评论数: 6 | 发布时间: 2025-02-04 11:05
文章摘要

**nv_cluster_lod_builder** 是一个连续细节层次（LOD）网格库，用于优化几何图形渲染。与传统离散LOD不同，它通过预计算原始网格的三角形簇，实现细节的平滑过渡。该库的主要特点包括： 1. **快速渲染**：根据相机位置动态调整几何细节。 2. **减少内存使用**：支持几何流式传输，特别适用于光线追踪。 主要流程包括： - **构建LOD**：通过递归地对三角形进行分组和简化，确保边界匹配。 - **选择簇**：根据相机距离选择合适的簇进行渲染。 - **空间层次结构**：优化运行时簇选择性能。 - **流式传输**：根据需要加载几何数据以节省内存。 输入是一个包含数百万三角形的面网格，输出是简化后的簇和其层次结构。该库还提供了构建集成和使用示例的指南。 详情和参考实现可访问：https://github.com/nvpro-samples/vk_lod_clusters

Alibaba/T-HEAD's Xuantie C910

作者: mfiguiere | 评分: 196 | 评论数: 11 | 发布时间: 2025-02-04 05:40
文章摘要

Alibaba旗下子公司T-HEAD推出的Xuantie C910是一款高性能RISC-V处理器核心，目标应用包括AI、边缘服务器、工业控制和高级驾驶辅助系统（ADAS）。C910是T-HEAD首个支持乱序执行的处理器，采用12nm工艺，主频可达2-2.5GHz，并在7nm工艺下可提升至2.8GHz。它支持RISC-V矢量扩展（RVV 0.7.1），具有低功耗、小面积的特点。C910每核心占用0.8平方毫米，动态功耗约为100微瓦/MHz。其前端设计能够处理16位和32位RISC-V指令及矢量扩展需求，包含64KB的指令缓存和16条目的全关联分支预测表。尽管部分源代码自动生成，增加了分析难度，但C910的开源特性允许更深入地研究其核心细节。

文章摘要

文章摘要

Httptap: View HTTP/HTTPS requests made by any Linux program

作者: alexflint | 评分: 570 | 评论数: 30 | 发布时间: 2025-02-03 16:28
文章摘要

**httptap** 是一个开源工具，用于查看任何 Linux 程序发出的 HTTP/HTTPS 请求。通过运行 `httptap -- ` 命令，用户可以捕获程序发出的请求和响应。例如，使用 `curl` 或 Python 发出的请求，工具会显示请求方法、URL、状态码和响应大小等信息。 ### 特点： - 无需 root 权限，无需设置守护进程或更改系统配置。 - 不会影响系统的 iptables 规则或路由表。 - 独立 Go 二进制文件，无依赖。 - 仅支持 Linux，依赖于 Linux 特有的系统调用（如网络命名空间），难以移植到其他操作系统。 ### 使用示例： 1. 捕获 `curl` 请求： ```bash httptap -- curl https://monasticacademy.org ``` 2. 捕获 Python 请求： ```bash httptap -- python -c "import requests; requests.get('https://monasticacademy.org')" ``` 3. 捕获 `gcloud` 请求： ```bash httptap -- gcloud compute instances list ``` 4. 捕获 `kubectl` 请求： ```bash httptap --https 443 6443 -- kubectl get all --insecure-skip-tls-verify ``` 工具提供了预构建的二进制文件和 Go 安装方式，方便用户快速使用。
