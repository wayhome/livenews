Httptap: View HTTP/HTTPS requests made by any Linux program

作者: alexflint | 评分: 410 | 评论数: 26 | 发布时间: 2025-02-03 16:28
文章摘要

`httptap` 是一个用于监控 Linux 程序发出的 HTTP/HTTPS 请求的工具。用户可以通过运行 `httptap -- ` 来查看任何 Linux 程序发出的请求和响应，无需 root 权限或对系统做任何更改。它是一个静态 Go 二进制文件，不依赖其他组件，也不会影响系统中其他进程。当前仅支持 Linux，因其使用了特定于 Linux 的系统调用（如网络命名空间），难以移植到其他操作系统。 使用示例包括监控 `curl` 或 `python` 发出的请求，以及查看 `gcloud` 和 `kubectl` 等命令行工具与 API 的交互。例如： ```bash $ httptap -- curl https://monasticacademy.org $ httptap -- python -c "import requests; requests.get('https://monasticacademy.org')" $ httptap -- gcloud compute instances list $ httptap --https 443 6443 -- kubectl get all --insecure-skip-tls-verify ``` 用户可通过预编译二进制或 Go 安装方式获取 `httptap`。
