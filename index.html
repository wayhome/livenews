文章摘要

SQLite Page Explorer 是一个小型GUI工具，用于以SQLite的页面级别浏览数据库文件，展示其在磁盘上的存储方式。该工具帮助开发者了解索引在磁盘上的实际结构、更紧凑的数据存储方式以及发现模式层面不易察觉的问题和低效之处。它基于redbean构建，是一个仅6.5 MB的可执行文件，支持在Windows、Linux、macOS和多种BSD系统上运行。用户可以通过拖放数据库文件或命令行方式使用该工具。需注意大文件加载可能较慢，并且由于可执行文件特性，可能会引发病毒警告。该项目代码开源，使用MIT许可证，欢迎贡献改进。

OpenLDK: A Java JIT compiler and runtime in Common Lisp

作者: varjag | 评分: 123 | 评论数: 11 | 发布时间: 2025-02-05 12:17
文章摘要

OpenLDK 是一个用 Common Lisp 实现的 Java 即时（JIT）编译器和运行时环境。它通过将 Java 字节码逐步翻译为 Lisp 代码，再编译为本地机器码来执行。这使得 Java 类可以无缝映射到 Common Lisp 对象系统（CLOS）的类，实现 Java 与 Lisp 的互操作。其主要特点包括：Java 字节码到 Lisp 的翻译、本地机器码编译、CLOS 集成以及利用 OpenJDK 的运行时库。OpenLDK 适用于在使用 SBCL（Steel Bank Common Lisp）时需要集成某个 Java 库的场景，无需独立的 Java 运行时环境。目前该项目主要在 Linux 下测试，并使用 SBCL 作为基础环境。

文章摘要

Frank Lloyd Wright, known for his anti-density, pro-suburb views and organic architecture philosophy, surprisingly proposed a mile-high skyscraper called "The Illinois." This tower, at 1,609 meters, was intended to encapsulate an entire city within one structure, satisfying density needs while preserving the surrounding landscape for his idealized Broadacre City of low buildings and green spaces. The tripod-like design and innovative "taproot" foundation were meant to handle wind forces and ensure stability. Wright's motivation may have been to critique the skyscraper race or to acknowledge humanity's inherent desire for vertical structures, even if it contradicted his own beliefs. Despite his anti-city stance, Wright's design demonstrates an engagement with the concept of urban density.
