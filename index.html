文章摘要

1988年，Macintosh Allegro Common Lisp (MACL) 发布了1.3版本，尽管LISP自1950年代以来存在，但未如C和Pascal等语言流行。LISP在人工智能领域和处理大型灵活数据方面表现出色。MACL 1.3具有优秀的调试器，并支持将LISP代码与Toolbox调用和对话框结合，创建合适的Macintosh用户界面。它包含2600个预编译的Common Lisp函数，并支持多种优化设置。 MACL 1.3引入了Object LISP扩展，支持多重继承和方法组合，未来将集成完整的Common Lisp对象系统(CLOS)。当前，用户可单独购买类似的PCL系统。 MACL提供高效的工作环境，包括设计良好的编辑器、对象检查器和调试器。其编辑器易于扩展和自定义，支持创建新命令并绑定键位。数据检查器可显示函数或变量的值，并支持递归检查。 调试器可显示变量和系统堆栈，帮助跟踪程序执行。用户可使用Allegro Interface Designer (AID) 创建Macintosh风格的界面，或直接调用Macintosh Toolbox构建自定义窗口。MACL还支持与Pascal和外部函数接口，提供强大灵活的开发环境。

Httptap: View HTTP/HTTPS requests made by any Linux program

作者: alexflint | 评分: 366 | 评论数: 24 | 发布时间: 2025-02-03 16:28
文章摘要

**httptap** 是一个用于查看 Linux 程序发出的 HTTP/HTTPS 请求的工具。通过运行 `httptap -- ` 命令，可以捕获并显示任何 Linux 程序发出的 HTTP 请求和响应，无需 root 权限或对系统进行广泛配置。它利用 Linux 的网络命名空间功能，不会影响系统中的其他进程，且为静态 Go 二进制文件，无依赖。目前仅支持 Linux，难以移植到其他操作系统。 示例： - 使用 `httptap -- curl https://monasticacademy.org` 可以捕获到 curl 命令的请求和308重定向响应。 - 使用 `httptap -- python -c "import requests; requests.get('https://monasticacademy.org')"` 可以查看 Python 程序发出的请求，包括重定向后的200响应。 - 还可以查看其他工具如 `gcloud` 和 `kubectl` 发出的 HTTP 请求。 安装方式包括通过预编译二进制文件或使用 Go 安装。

评论摘要

这篇文章介绍了一项将视频制作成趣味翻翻书的服务。用户可以上传视频（支持MP4、AVI、MOV等多种格式），选择封面颜色并实时预览，最后订购高质量印刷版翻翻书。该产品适合作为礼物或纪念品，特别适用于婚礼、活动纪念等场合。目前提供全球免费 shipping 的限时优惠，单本价格从25美元起。此外，支持批量定制，可添加公司或活动品牌标识，并有快速交付选项。整个制作过程简单，支持手机端操作，用户可以轻松创建和分享他们的独特视频记忆。
