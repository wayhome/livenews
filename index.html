Smallpond – A lightweight data processing framework built on DuckDB and 3FS

作者: overflowcat | 评分: 248 | 评论数: 10 | 发布时间: 2025-02-28 01:56
文章摘要

**smallpond** 是一个轻量级数据处理框架，基于 DuckDB 和 3FS 构建。其主要特点包括：高性能数据处理、可扩展至 PB 级数据集、无需长期运行的服务，易于操作。支持 Python 3.8 至 3.12，安装命令为 `pip install smallpond`。使用示例展示了如何下载数据、初始化会话、读取和处理数据以及保存结果。该框架已在 GraySort 基准测试中进行评估，在包含 50 个计算节点和 25 个存储节点的集群上，30 分 14 秒内处理了 110.5TiB 数据，吞吐量达 3.66TiB/分钟。项目基于 MIT 许可，贡献者包括 Wang Runji 和 Liu Yiyuan。
