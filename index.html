Jellyfin: The Free Software Media System

作者: doener | 评分: 184 | 评论数: 28 | 发布时间: 2025-02-15 22:39
文章摘要

Jellyfin是一个免费的开源媒体系统，允许用户收集、管理和流式传输各种媒体内容。它支持电影、电视剧、音乐、直播电视、书籍、照片等，并提供远程共享观影功能（SyncPlay）。Jellyfin可在多种平台上使用，包括网页、桌面、Android、Apple、Amazon、Roku和Kodi等。该软件注重隐私，不追踪用户数据，也没有隐藏费用。用户可以通过官方和第三方客户端随时随地访问自己的媒体库。Jellyfin由社区志愿者开发，并基于GNU GPL许可证发布，用户可以自由使用、修改和分发。

Watt The Fox?

文章摘要

本文描述了作者在新笔记本上发现的一个问题：扬声器在无声音播放时会发出微弱的白噪声。通过调查，作者使用Linux命令`pactl list sinks short`发现音频 sink 处于 IDLE 或 RUNNING 状态时会出现白噪声，即使没有播放声音。进一步调查显示，Firefox 是导致该问题的程序，即使标签页静音，仍会导致音频 sink 保持活跃状态，特别是来自 Outlook 网页版的通知声音。这一问题不仅令人厌烦，还会增加 CPU 使用率和电池消耗。作者通过重新加载相关标签页或重启 Firefox 来解决此问题。

评论摘要

Schemesh: Fusion between Unix shell and Lisp REPL

作者: cosmos0072 | 评分: 77 | 评论数: 6 | 发布时间: 2025-02-15 19:00
文章摘要

Schemesh 是一个将 Unix shell 与 Lisp REPL 结合的交互式 shell，旨在替代传统的 Unix shell（如 bash、zsh 等）。它支持 Unix shell 的常见功能，如命令执行、重定向、管道、作业控制等，同时集成了 Chez Scheme 支持的完整 Lisp REPL。用户可以在 Schemesh 中直接输入命令作为 Unix shell 使用，或输入 Lisp 表达式进行编程。Schemesh 可以在两种语法间无缝切换，支持高级作业管理，如启动、重定向和控制 Unix 进程。通过 Lisp 语法，用户还可以操作由 Unix shell 语法创建的作业对象。Schemesh 提供了强大的作业控制机制，如中断、挂起、前后台恢复作业等。相比传统 shell 脚本，Schemesh 提供了更高效、灵活的 Lisp 环境用于脚本和编程任务。

评论摘要
