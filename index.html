Privacy Pass Authentication for Kagi Search

作者: b3n | 评分: 488 | 评论数: 39 | 发布时间: 2025-02-13 19:57
文章摘要

Kagi Search introduces Privacy Pass, a new privacy feature based on an authentication protocol standardized by the IETF. Privacy Pass allows users (Clients) to authenticate to Kagi (Server) while ensuring the server can verify access rights without identifying the specific user or their search history. This enhances privacy, particularly for users of Kagi’s paid search services, by making tracking technically impossible. Initially, this feature is available on Professional, Ultimate, Family, and Team plans but not on Trial and Starter plans due to technical limitations. Privacy Pass operates in two phases: token generation and token redemption. During token generation, users prove their right to generate tokens, typically by presenting a Kagi session cookie. These tokens are indistinguishable and untraceable. In the token redemption phase, users access services using these tokens, with the server unable to link tokens to specific users. Users can enable or disable Privacy Pass as needed, choosing between full-featured mode with normal privacy or maximum privacy with core features. The implementation is open source and integrated into the Orion Browser, Kagi for Android, and available as an extension for Firefox and Chrome. Safari support is not yet available.

文章摘要

文章介绍了通过计算机模拟重现眼睛进化的过程，类似于寒武纪生命大爆发的视觉演化。研究团队创建了一个虚拟培养皿，从单个感光细胞开始，模拟生物在数百万年间进化出不同类型眼睛的过程。这些数字生物根据生存需求进化出不同类型的眼睛，如用于导航的复眼和用于辨别食物与毒素的相机眼。模拟还展示了随机基因突变如何影响视觉进化，以及透镜等结构如何自然出现以应对生存压力。该项目旨在研究自然界未发生的假设情景，理解生物视觉进化的原理，并探索利用这些原理设计更智能的人工视觉系统。团队还开发了一个模拟器，允许用户定义和进化自己的生物体，以研究视觉和智力进化。

评论摘要
主要讨论点：进化论和眼睛演化的解释 不同观点： • 一些评论者（如bloomingkales）支持通过Dawkins的解释来理解眼睛的进化，并分享了相关的视频资源。该视频由理查德·道金斯（Richard Dawkins）讲解，通过简明的描述帮助观众理解眼睛这样复杂的器官是如何通过自然选择逐步演化而来的。 • 另一部分评论者对眼睛演化的复杂性提出质疑。他们认为，即使有逐步演化的解释，眼睛涉及的光学、神经学等多个系统的同步演化仍然让人难以完全信服。有人提到，眼睛的复杂性似乎更符合“智能设计”的论点，而非纯粹的自然选择结果。 • 还有评论者指出，视频中的解释虽然清晰，但过于简化，忽略了一些重要的生物学细节。例如，不同物种的眼睛结构和演化路径存在巨大差异，单一的解释可能无法涵盖所有情况。 补充讨论： • 一些支持者进一步补充了其他资源和文献，以强化道金斯的观点，强调了自然选择在不同物种中对眼睛演化的实际证据。 • 争议的焦点集中在眼睛演化的可信度上。批评者认为复杂器官的同步演化需要更多的解释和证据，而支持者则坚持认为，累积选择等机制足以解释这种复杂性。 • 此外，部分评论提到了哲学层面的讨论，质疑科学解释是否足以涵盖所有生命复杂性，尤其是涉及人类感知和意识等更抽象的领域。

主要讨论点：对多样性、公平与包容（DEI）政策及其取消的讨论 不同观点： • **delichon**：提到了一项行政命令，要求合同或资助接收方遵守联邦反歧视法律，并证明其不运营任何违反这些法律的DEI项目。该评论强调了政策层面对DEI的限制。 • **dpedu**：认为DEI并非真正的努力，否则不会被大公司迅速取消。如果DEI对公司真正有利，就不会被迅速取消。 • **mmooss**：提出通过数学或逻辑概念解决偏见问题，主张消除所有偏向某一特定群体的项目和流程，而不仅仅针对弱势群体。建议取消所有群体的平权措施和DEI，以实现真正的公平正义。 • **yapyap**：认为公司并不真正关心DEI，只是为了迎合当前的领导者。 • **guywithahat**：认为人工智能应在政治上保持中立，取消DEI是好事，因为早期问题源于AI因政治原因拒绝回应某些提示。 • **bayarearefugee**：主张招聘应严格基于能力，并用讽刺的方式列举了一些反例，说明如果基于其他标准招聘会导致问题。 • **mc32**：认为公司本就不应宣传DEI，因为这表明原本没有公平招聘。DEI项目往往是管理层为了自我感觉良好的花钱项目。 • **poszlem**：强烈反对DEI，认为公司迅速屈从于强权者的意愿令人震惊，并联想到历史上的极权主义例子。 补充讨论： • **DEI的有效性和必要性**：一些评论（如dpedu和mmooss）质疑DEI的真正效果和必要性，认为如果它真的有效，不会被迅速取消。 • **政治与商业的关联**：yapyap和poszlem提到公司迎合政治领导者的现象，并将其与历史上的极权主义行为相联系。 • **招聘的公平性**：bayarearefugee和mc32强调招聘应基于能力，而不是其他标准，并批评了公司通过DEI项目自我标榜公平的做法。 争议焦点： • **DEI政策的真实效果和目的**：一部分人认为DEI是必要的平权措施，而另一部分人则认为它是表面的、无效的，甚至是有害的。 • **招聘公平性的实现方式**：评论中对如何实现真正的招聘公平性存在不同看法，包括是否需要DEI项目或完全取消这些项目。
